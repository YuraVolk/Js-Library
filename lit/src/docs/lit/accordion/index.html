Accordion

The accordion component represents multiple options that user may pick to read more about, initially being able to see only headings. This accordion works only with two components working in tandem: accordion-component and accordion-item-component. These two components through a context communicate with each other and track the updated index, with the parent component holding the current selected index(es). Through this, the accordion items can fully isolate max height and animation logic. Here you can see an example of this component:
