Gallery Carousel

The gallery carousel component represents a 2D carousel that allows items to be cycled through the visible area, showing one item at a time by default. This is a classic and most common carousel type, which is commonly seen on websites, and has toggles and arrows to navigate in the carousel.

  • smoothDiametralTransition: Determines whether the transition between the last item and the first item is smooth. By default, this property is set to false.
  • current: Specifies the index of the current item being displayed in the carousel.
  • frameGap: Defines the gap between each item in the carousel.
  • animationDuration: Sets the duration of the animation when transitioning between items.
  • showArrows: Determines whether the arrows for navigating to the previous or next item are displayed. False by default.
  • showToggles: Specifies whether the toggles for switching between items are shown. False by default.
