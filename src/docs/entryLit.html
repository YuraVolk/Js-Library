Advanced Password Validation on JS

Password validation is used in almost every single registration. It prevents user from typing too easy and uncomplicated password. It is mostly based on iteration and regex-es.

USAGE:
1. Change classes !important
2. Delete unused tests
3. Change test requirements
4. Add another tests (if needed)

Has at least 6 symbols

Contains special character (_, %, ^, et.c.)

Has both numbers and letters

Doesn't use common pattern (Qwerty, et.c)

Has at least 2 upper case letters
